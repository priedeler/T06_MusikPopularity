# T06 MusikPopularity

Project analyzing the popularity of music using Machine Learning.

## Overview
This project contains a Jupyter Notebook (`MusikPopularity.ipynb`) that:
1. Processes a large music dataset.
2. Performs data balancing and filtering.
3. Trains and evaluates Naive Bayes models (Gaussian and Categorical).
4. Visualizes the distribution of track popularity and model performance.

## Files
- `MusikPopularity.ipynb`: Main analysis notebook (cleaned of Google Colab specific code).
- `final_processed_dataset.csv`: Processed dataset used for training.
- `final_processed_dataset_V2.csv`: Balanced and filtered dataset.
- `requirements.txt`: Python dependencies.

## Getting Started
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the Jupyter Notebook:
   ```bash
   jupyter notebook MusikPopularity.ipynb
   ```

*Note: The raw dataset `master_dataset_enriched.csv` is not included in the repository due to its size (~178MB).*
